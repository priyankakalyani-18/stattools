# stattools

An R package for computing summary statistics with outlier detection, 
min-max normalization, and visualization of numeric data.

## Installation

```r
devtools::install_github("priyankakalyani-18/stattools")
```

## Functions

- `compute_summary()` — computes descriptive statistics and detects outliers
- `normalize_data()` — performs min-max normalization
- `plot_summary()` — visualizes data with outliers highlighted using ggplot2

## Dataset

Includes `sensor_data` — a synthetic dataset of sensor readings from a 
materials testing experiment.
